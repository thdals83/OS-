# OS-Scheduler
OS 스케쥴러 간트차트 구현

## 팀원
  - 송민혁
  - 김지훈

## 개발 언어
   - JAVA
   
## 라이브러리
   - AWT & SWING
   
## 프로그램 구현 기능
    - 프로세스의 개수와 ID, 도착시간, 서비스 시간, 우선순위, 시간 할당량을 입력받는 창
    - 각 스케줄링(FCFS, SJF, 비선점 Priority, 선점 Priority, PR, SRT, HRN)의 결과를 나타내는 창으로 구성
    - 각 프로세스별 응답시간, 평균 응답시간, 각 프로세스별 반환 시간, 평균 반환 시간을 간트차트로 제공
